# Travenner

Travenner is a travel planning web application designed to help users plan trips, discover destinations, and organize itineraries in one place. It allows users to explore travel options, save plans, and manage their journeys easily.

The project is built to be lightweight, scalable, and user-friendly for travelers who want a simple way to plan and track their trips.

## Features
- Explore travel destinations and attractions  
- Create and manage trip itineraries  
- Save favorite places and plans  
- Track travel schedules and bookings  
- User profiles and trip history  

## Tech Stack
- Frontend: HTML, CSS, JavaScript / React  
- Backend: Node.js / Java / Python  
- Database: MySQL / PostgreSQL / MongoDB  
- API: RESTful HTTP endpoints  

## Purpose
This project is intended for learning, portfolio development, and building practical full-stack applications focused on travel planning and user experience.
